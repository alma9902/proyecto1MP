Proyecto 1 de Modelado y Programación
Utilice la herramienta build para probar :
	go build github.com/alma9902/proyecto1MP
Para entrar como servidor, entonces deberá indicar con una bandera
	./proyecto1MP --mode server puerto
para entrar como cliente entonces deberá indicar el ip, el puerto
y el nombre
	./proyecto1MP --mode client ip puerto nombre
